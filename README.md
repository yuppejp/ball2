# ball2
HTML+JavaScriptによるボール転がしのサンプル(加速度を使ったサンプル)
加速度センサーで等加速度直線運動をシミュレーションします。
ボールがなめらかに描画されるようにrequestAnimationFrameを使用しました

## 使用方法:
* PHPのビルトインサーバを起動  
  ``$ php -S localhost:8000``

* ブラウザでアクセス  
  ``http://localhost:8000/``
